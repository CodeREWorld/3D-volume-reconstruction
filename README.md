# 实时三维体积重建算法

小容灵采用实时三维体积重建算法，完成车辆容积的准确获取


## 环境配置

Requirements:
- DirectX SDK June 2010
- NVIDIA CUDA 7.0


Optional:
- Kinect SDK (2.0 and above)
- Prime sense SDK
